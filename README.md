# 👥 Previsão de Headcount

> Previsão de necessidade de headcount com IA. Projeta demanda de pessoal por área alinhada ao crescimento, otimizando o planejamento de quadro.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/previsao-headcount/

[![Vertical](https://img.shields.io/badge/Vertical-RH-blue)]() [![IA](https://img.shields.io/badge/IA-ML-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**90%** Acurácia · **todas** Áreas · **12 meses** Horizonte · **otimizado** Custo

## 🧠 Tecnologias de IA
ML · Forecasting · Workforce Planning

## ✨ O que faz
- Projeção de quadro por área
- Alinhamento ao crescimento
- Cenários de contratação
- Otimização de custo de pessoal

## 🗄️ Fonte de dados real
[RH + crescimento](https://www.gov.br/trabalho)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com RH + crescimento
2. **Processamento com IA** — ML · Forecasting · Workforce Planning
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
